1- Clonar repositorio
    -> git clone URL:HTTPS
2- Confirmar archivos clonados
    -> ls -al
3- Realiza un cambio en un archivo
    -> vim archivo.txt
4- Agregar cambios
    -> git commit -am "dejar mensaje"
5- Enviar cambios
    ** Antes de enviar el integrante debe terner acceso al repositorio
    -> git pull origin master
    -> git push origin master
